Скрипт для работы с файлом Бары.json.
С его помощью можно определить:
	1.Самый большой бар
	2.Самый маленький бар
	3.Самый близкий бар
	
Для правильной работы скрипта необходимо указать путь к файлу Бары.json.
Файл Бары.json можно скачать по этой ссылке http://data.mos.ru/opendata/export/1796/json/2/1
Для поиска самого близкого бара вам необходимо указать ваши координаты.
